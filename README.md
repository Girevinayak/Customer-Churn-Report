# Customer Churn Report

## Overview
This Power BI project analyzes customer churn and identifies high-risk customers.  
It helps businesses track the number of customers at risk, actual churn, and churn rates in an easy-to-understand dashboard.

---

## Goal
- Identify high-risk customers  
- Track actual churn and churn rate  
- Provide quick insights for management to take action

---

## Data
The dataset includes:  
- Customer ID (anonymized)  
- Risk status (High / Medium / Low)  
- Churn information (whether the customer left)

> ⚠️ Note: Real customer data is anonymized to protect privacy.

---

## Key Metrics
- Total Customers  
- High Risk Customers  
- High Risk %  
- Customer Churn Distribution  
- Sum of Actual Churn  
- Churned Customers  
- Churn Rate

---

## Visuals
- Cards for each metric  
- Pie chart for churn distribution  
- Optional line chart for trends over time

## How to Use
1. Open the `.pbix` file in Power BI Desktop  
2. Explore the cards and charts to understand high-risk customers and churn trends  
3. Optionally, update the dataset or add filters for more detailed analysis
