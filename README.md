# Customer Churn Report

## Goal
Track high-risk customers and churn metrics using Power BI.

## Key Metrics
- Total Customers
- High Risk Customers
- High Risk %
- Churned Customers
- Churn Rate

## Visuals
- Cards for each metric
- Pie chart for churn distribution

## How to use
Open the `.pbix` file in Power BI Desktop to explore the dashboard.
